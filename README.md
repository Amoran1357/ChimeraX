# CHIMERAX Ω

Autonomous Multi-Agent Trading Intelligence System built for the BNB Hack: AI Trading Agent Edition.

## Overview

CHIMERAX is a self-custody AI trading system that combines:

- CoinMarketCap Market Intelligence
- Multi-Agent Elder Council
- Consensus-Based Trade Decisions
- Evolution Engine
- Trust Wallet Agent Kit Execution
- BNB Chain Settlement

## Architecture

CoinMarketCap Agent Hub
↓
Market Intelligence Layer
↓
Digital Elder Council
(Momentum, Narrative, Whale, Risk, Chaos)
↓
Consensus Engine
↓
Evolution Engine
↓
Trust Wallet Agent Kit
↓
BNB Chain

## Features

### Market Intelligence Layer
Aggregates market data, sentiment, narratives and volatility signals.

### Elder Council
Five specialized AI agents independently analyze markets.

### Consensus Engine
Weighted voting system producing final trade decisions.

### Evolution Engine
Stores prior decisions and adapts future behavior.

### Execution Layer
Trust Wallet Agent Kit powered self-custody execution.
